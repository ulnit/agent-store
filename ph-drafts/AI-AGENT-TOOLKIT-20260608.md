>
> Every tool outputs clean, structured CLI text designed for AI agent ingestion. Under 200 lines per tool. Pipeable stdin/stdout. Cross-platform (Linux/Mac/Windows).
>
> Built to run on a $35 Raspberry Pi — perfect for 24/7 AI automation with zero cloud costs. Power-pairs with AI API Gateway, AI Video Factory, and Agent Templates for a complete AI infrastructure stack on a single Pi.

> I built this because I was tired of installing curl|jq|pup|entr just to do basic agent automation tasks. Turns out Python's stdlib can handle all of it — web scraping, JSON querying, file watching — with zero dependencies. 😅
>
> **What's in the box:**
> - 🌐 `web-fetch` — Clean text extraction from any URL (strips HTML/JS/CSS)
> - 📋 `json-query` — Dot-notation JSON queries (no more jq syntax wrestling)
> - 👁️ `file-watch` — Auto-trigger scripts on file changes (like entr, but Python)
> - 📊 20+ more tools for scraping, APIs, data, and file ops
>
> **The crazy part:** Every tool is under 200 lines. stdlib only. No pip install needed for the core — just `curl | bash` and you're running.
>
> **Use it for:**
> - 🤖 AI agent pipelines — structured CLI output for AI ingestion
> - 💰 AI API reselling — scrape competitor pricing, monitor APIs
> - 🎬 AI video automation — trigger ffmpeg pipelines on new files
